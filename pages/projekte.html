PROJEKTE

MY-CHECK-IN

Dauer des Projekts: 6 Monate

Kurzbeschreibung

Entwicklung eines Check-In Systems für ein Hotel, welches Besuchern das Einchecken über ein „Self-Service-Terminal“ ermöglicht. Das System besteht aus einer App, die im Kiosk-Modus auf einem Android-Tablet im Terminal läuft, einem Content Management System zur Verwaltung der Zimmer sowie aus einer Schnittstelle, welche mit der App und dem CMS kommuniziert. Zusätzlich wurde eine Software für einen Kartendrucker entwickelt, welcher die Check-In Informationen für die Gäste nach dem Check-In-Vorgang auf eine Plastikkarte druckt.

Tätigkeiten und Aufgaben

Haupttätigkeit war die Entwicklung einer hybriden App mit Ionic 3 und einer webbasierten Admin-Oberfläche in Javascript / Typescript (Angular), HTML5 und CSS3. Zusätzlich die Entwicklung des Backends in Node mit dem Express Framework mit JSON-REST Schnittstelle. Das Projektmanagement erfolgte mit Trello und als Versionsverwaltungs-Tool wurde Git eingesetzt.

SAM (Software Asset Managment)

Dauer des Projekts: 4 Monate

Kurzbeschreibung

Ein webbasiertes Lizenzmanagement-System für den B2B Bereich. Die Webanwendung ermöglicht es Lizenzgebern, firmeneigene Lizenzen zu verwalten. Dazu zählt unter anderem die Anlage, Bearbeitung und Freischaltung von Unternehmen & dazugehöriger Mitarbeiter sowie die Vergabe von Lizenzen, Prüfung auf Gültigkeit und Verwaltung von lizenzbasierten Produkten (z.B. Systeme, Produkte, Software) sowie damit zusammenhängender Module.

Besonderheit

Die Initiative Mittelstand verleiht den INNOVATIONSPREIS-IT 2018 an besonders innovative IT- Lösungen mit hohem Nutzen für den Mittelstand. Das Produkt erhielt im April 2018 in der Kategorie „IT-Service“ von der Fachjury das Prädikat „BEST OF 2018“.

Tätigkeiten und Aufgaben

Haupttätigkeit war die Entwicklung des Front-Ends mit Typescript (Angular), HTML5, CSS3 und die Einbindung des Java Backends (Spring Framework) über eine REST Schnittstelle. Zusätzlich wurde die REST Schnittstelle mit konzipiert und die komplette Anwendung getestet. Das Projektmanagement erfolgte mit JIRA und als Versionsverwaltungs-Tool wurde Git eingesetzt.

AUKTIONSPLATTFORM

Dauer des Projekts: 6 Monate

Kurzbeschreibung

Auktionsplattform, in der Besucher auf sogenannte „Penny“- Auktionen bieten können, wodurch sich der Preis und die verbleibende Zeit der Auktion automatisch erhöht. Es wurde eine hoch-performante Plattform geschaffen, welche durch Web-Sockets die Server-Zugriffe auf ein Minimum reduziert und dadurch mehreren tausend Nutzern gleichzeitig standhält. Zusätzlich wurde ein CMS entwickelt, mit welchem die Verwaltung der Inhalte wie Produkte, Auktionen und User ermöglicht wird.

Tätigkeiten und Aufgaben

Haupttätigkeit war die Entwicklung des Front-Ends, welches aus der grafischen Benutzeroberfläche für Bieter und einer Admin Oberfläche besteht. Entwickelt wurde mit Typescript (Angular), Node.JS, HTML5 und CSS3 (Material Design). Das Frontend ist angebunden an ein Java Backend (Spring Framework) mit einer JSON REST Schnittstelle. Das Projektmanagement erfolgte mit JIRA und als Versionsverwaltungs-Tool wurde Git eingesetzt.

REZEPTFREI.INFO

Dauer des Projekts: 2 Monate

Kurzbeschreibung

Webseite mit dem Ziel, den Besuchern eine schnelle und einfache Navigation zu qualitativ hochwertigen rezeptfreien Medikamenten anzubieten (zu bestimmten Indikationen nach der Auswahl eines entsprechenden Körperbereichs). Dem Verbraucher wird die Wahl überlassen, ob und wie er sich zu vorgeschlagenen Produkten anderweitig informiert, sei es auf der Hersteller/Produktseite, dem Testbericht der Stiftung Warentest oder direkt bei der Apotheke, wo das Produkt erworben wird.

Besonderheit

Die Initiative Mittelstand verleiht den INNOVATIONSPREIS-IT 2017 an besonders innovative IT- Lösungen mit hohem Nutzen für den Mittelstand. Das Produkt erhielt im März 2017 in der Kategorie „E-HEALTH“ von der Fachjury das Prädikat „BEST OF 2017“.

Tätigkeiten und Aufgaben

Entwicklung des Frontends mit Javascript / Typescript (Angular), HTML5, CSS3, jQuery und Anbindung an das node Backend (Restify Framework) über eine JSON-REST Schnittstelle.

TARIFDATENBANK

Dauer des Projekts: 18 Monate

Kurzbeschreibung

Eine Webanwendung zum Verwalten der Preise für professionelle Gutachten in der Logistikbranche. Die Software wurde mit Berücksichtigung des ISO 9897:1997 Standards für den Austausch von Containerdaten (CEDEX) entwickelt.

Tätigkeiten und Aufgaben

Haupttätigkeit war die Entwicklung des Frontends mit Javascript (React.JS), HTML5 und CSS3 (Bootstrap). Zusätzlich wurde das komplette Node Backend (Express Framework) entwickelt und angebunden. Genutzt wurde eine MySQL Datenbank & JSON REST API. Das Projektmanagement erfolgte mit JIRA und als Versionsverwaltungs-Tool wurde Git eingesetzt.
