PROJEKTE

MY-CHECK-IN

Dauer des Projekts: 6 Monate

Kurzbeschreibung

Entwicklung eines Check-In Systems für ein Hotel, welches Besuchern das Einchecken über ein „Self-Service-Terminal“ ermöglicht. Das System besteht aus einer App, die im Kiosk- Modus auf einem Android-Tablet im Terminal läuft, einem Content Management System zur Verwaltung der Zimmer sowie aus einer Schnittstelle, welche mit der App und dem CMS kommuniziert. Zusätzlich wurde eine Software für einen Kartendrucker entwickelt, welcher die Check-In Informationen für die Gäste nach dem Check-In-Vorgang auf eine Plastikkarte druckt.

Tätigkeiten und Aufgaben

Haupttätigkeit war die Entwicklung einer hybriden App mit Ionic 3 und einer webbasierten Admin-Oberfläche in Javascript / Typescript (Angular), HTML5 und CSS3. Zusätzlich die Entwicklung des Backends in Node mit dem Express Framework mit JSON-REST Schnittstelle. Das Projektmanagement erfolgte mit Trello und als Versionsverwaltungs-Tool wurde Git eingesetzt.
